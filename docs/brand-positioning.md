## Chosen name: Wavepilot

**Rationale:** "Wave" = trending content wave, always in motion. "Pilot" = the user is in control, navigating with confidence. Friendly and memorable — works for creators and SMBs equally. Not intimidating to non-marketers.

**Domain target:** wavepilot.co  
**Working title was:** TrendPilot (retired)

### Runner-up names considered
| Name | Verdict |
|---|---|
| TrendPulse | Clear and safe but slightly generic |
| Signalflow | Skews too technical |
| SpikeHQ | Creator-coded, punchy — keep as alt if wavepilot.co unavailable |
| Wavemark | Balanced, slightly more SMB-coded |

## Target users

**Primary A — The Creator**  
Solo content creators (TikTok, YouTube, Instagram). 5K–100K followers. Knows they need a strategy but wastes hours on trend research. Wants to know what to post *before* it peaks. Tech-comfortable.

**Primary B — The Small Business Owner**  
Runs their own social media with no marketing background. Posts randomly, sees little return. Wants a simple, clear plan without needing to become a "content person." Phone-first.

## Brand voice

| Principle | In practice |
|---|---|
| Encouraging, not overwhelming | "Here's exactly what to post this week" — not "leverage omnichannel synergies" |
| Confident, not salesy | Tells users what works and why — doesn't overpromise |
| Direct, not dumbed down | Short sentences, no filler. Respects the user's intelligence. |
| Warm, not casual | Knowledgeable friend — not a bro, not a corporate bot |

### Voice don'ts
- No marketing buzzwords: "leverage", "synergy", "omnichannel", "optimize"
- No excessive enthusiasm: "Amazing!", "So exciting!", "You've got this!"
- No hedging: "It might be helpful to perhaps consider..."
- No robotic formality: "Per your request, I have generated the following..."

## Visual direction

### Color palette
| Role | Hex | Usage |
|---|---|---|
| Primary | `#C84B24` | CTAs, active states, accents, logo |
| Primary hover | `#A73C18` | Hover states on primary elements |
| Primary deep | `#802E14` | Pressed states, emphasis text on light bg |
| Primary light | `#FCEEE8` | Subtle background tint for callouts, selected chips |
| Page background | `#FAFAF9` | Page surface (warm off-white) |
| Card background | `#FFFFFF` | Card, modal, input surfaces |
| Text primary | `#1F1F1F` | Headings, body copy |
| Text muted | `#7A808C` | Secondary copy, sub-hero, helper text |
| Border default | `#DDDDDD` | Card borders, dividers |
| Border input | `#E4E5EA` | Input borders |

### Typography
- **Display:** Red Hat Display (400, 500) — h1–h3, logo wordmark, pricing numbers
- **Body / UI:** Inter (400, 500, 600) — everything else
- **Code / mono:** `ui-monospace, SFMono-Regular, Menlo, monospace`
- Retired: Plus Jakarta Sans, DM Mono

### Design principles
- Warm and editorial — paper-like, not clinical
- Burnt orange is the only accent — used sparingly for emphasis
- Borders over shadows — the canvas relies on 1px dividers for structure
- Motion is restrained — 150–400ms eases, respect reduced-motion preferences
- Mobile-first — many SMB users access on phone
- Speed first — skeleton loaders, no blocking UI

## Competitive positioning

| Competitor | Gap Wavepilot fills |
|---|---|
| Buffer / Later | Scheduling tools — no trend intelligence, no AI strategy |
| Hootsuite | Enterprise-priced, overwhelming for solo users |
| Generic AI chatbots | No live trend data, no platform-specific output, no structure |
| Manual trend research | Hours of scrolling, always late to trends |

**Wavepilot's moat:** Cross-platform trend synthesis + niche-specific AI personalization + actionable output (not a report, a ready-to-use plan).
